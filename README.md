## ローカル環境で確認する

まず下記のコマンドで依存するプログラムをインストールします。

```bash
$ npm i
$ npm run install
```

下記のコマンドで開発サーバーが立ち上がります。

```bash
$ npm start
...
...
Starting server ...
Serving book on http://localhost:4000
```

`http://localhost:4000` にアクセスすることで表示の確認ができます。
